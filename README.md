# Zoom Agentic AI

Agentic AI for Automated Question Generation from Zoom Meeting Transcripts.

This repository is currently scaffolded for implementation after approval of the planning documents.

## Main Areas

- `backend/`: FastAPI backend, services, LangGraph workflow, database layer, integrations.
- `frontend/`: React + Vite dashboard.
- `database/`: schema and migration support files.
- `docs/`: approved specification and supporting documentation.
- `tests/`: backend, frontend, integration, performance, and security tests.
